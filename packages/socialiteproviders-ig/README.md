# socialiteproviders/instagram

Socialite Providers是一個擴展自Laravel Socialite的OAuth 1 & 2包集合。 其中的Manager包使用觀察者模式來擴展Socialite， 這種機制允許除了Laravel Socialite提供的認證提供者之外的多個其它提供者被使用， 且只有添加到監聽器的提供者被使用。

更新：從2019年10月15日開始，Instagram API平台上的新客戶端註冊和權限審查將停止，轉而使用Instagram Basic Display API。
